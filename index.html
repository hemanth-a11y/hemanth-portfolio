Projects

Railway-Style Clock Display

This is a custom-designed clock inspired by railway station boards. It features a pixel-style train engine animation that pulls the time across a 64x32 LED matrix. The clock includes a real-time date at the bottom and a "Platform No" title at the top. An ESP32 with GPS ensures accurate time, and the display is customizable via a web interface. Synchronization across multiple displays is possible using ESP-NOW protocol.

Tech: ESP32, LED Matrix, GPS, Arduino, ESP-NOW, Web Server

Gesture-Controlled Dodge Game

A real-time game built with Python where the player uses hand gestures captured via webcam to dodge obstacles falling from above. It uses OpenCV and MediaPipe to track hand movements and integrates with a game engine (Pygame) for interaction, scorekeeping, and game loop logic.

Tech: Python, OpenCV, MediaPipe, Pygame

Arduino Robotic Arm

This robotic arm is powered by an Arduino Nano and MG90S servos. It interprets hand gestures using a Python-based webcam tracker and translates them into servo movements. Designed for educational purposes, it helps understand inverse kinematics and gesture-based control systems.

Tech: Arduino Nano, MG90S, Python, OpenCV

ESP32 Smart Lock

A compact smart locking system that allows remote control of a physical lock using an ESP32 microcontroller. It features basic authentication logic, a relay to control the lock mechanism, and a future-proof design for adding RFID or keypad access. Can be controlled via a web interface hosted on the ESP32.

Tech: ESP32, Relay Module, Web Server, Arduino

ESP32 Digital Clock

This project is a simple but reliable digital clock made using an ESP32 and a display module. It shows real-time hours, minutes, and seconds. It was an early prototype for the more complex railway-style clock.

Tech: ESP32, LED Display, Arduino

Cinewhoop FPV Drone Build

A 4S-powered 3-inch Cinewhoop drone with a SpeedyBee BEE35 frame. Built for stable indoor and outdoor FPV flights, it uses SpeedyBee F4 flight controller, TX800 VTX, and Radiomaster ExpressLRS receiver system. Inspired by YouTube builds, it balances DIY customization with professional performance.

Tech: SpeedyBee BEE35, TX800, F4 FC, ExpressLRS, RP1, Radiomaster Pocket

Jarvis-Style Python Voice Assistant

A personal voice assistant coded in Python featuring offline support, custom male voice, and dark-themed GUI. It can execute basic commands, respond to queries, open applications, and control system functions. Built with ~600 lines of code, it is fully customizable and works without an internet connection.

Tech: Python, Tkinter, pyttsx3, SpeechRecognition, Offline Mode

Agricultural Robot

An advanced multi-purpose agricultural robot equipped with a NodeMCU (ESP8266) for wireless communication. It can perform tasks like cutting grass, sowing seeds, spraying water, and leveling soil. The robot is programmable, making it ideal for automating basic agricultural operations.

Tech: NodeMCU ESP8266, Automation, Agricultural Tools

Bluetooth Controlled Car

A remotely operated car built using Arduino Uno, L293D motor driver, and a Bluetooth module. The Bluetooth module receives commands from a smartphone or controller and sends them to the Arduino, which processes and transmits the instructions to the motor driver to control movement.

Tech: Arduino Uno, L293D Motor Driver, Bluetooth Module

Home Automation

A creative and functional home automation system using NodeMCU (ESP8266) and relay modules. It allows users to control household appliances like lights and fans through a smartphone or web dashboard over Wi-Fi. The system integrates real-time switching, device status feedback, and can be scaled with sensors for smart lighting, temperature control, and more. It's a step toward building a fully connected smart home environment.

Tech: NodeMCU ESP8266, Relay Module, Web Dashboard, IoT
